## 🌿 GreenDen - Plant Shopping Website

GreenDen is a responsive plant shopping website built using HTML and Tailwind CSS.
It is designed as an online platform to showcase and sell indoor and outdoor plants.

## 🚀 Live Preview
 https://dharshika1112007.github.io/Greenden_TailwindCSS/

## 📌 Features

- 🌿 Responsive Navigation Bar
- 🏡 Beautiful Hero Section
- 📖 About Us Section
- 🛍️ Product Listing Page
- 🔍 Search Bar UI
- 💬 Customer Reviews Section
- 📧 Newsletter Subscription UI
- 📞 Contact Form Page
- 📱 Fully Responsive Design (Mobile + Desktop)
- 🎨 Built using Tailwind CSS Utility Classes


## 🛠️ Tech Stack

- HTML5
- Tailwind CSS (CDN Version)
- Responsive Design Principles
